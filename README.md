### Installation

#### Install via npm

```
npm i leaflet.pm-measure-path
```

#### Include as ES6 Module

```js
import 'leaflet.pm-measure-path';
import 'leaflet.pm-measure-path/dist/leaflet.pm-measure-path.css';
```
